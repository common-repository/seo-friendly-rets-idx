Convert your RETS feed into an SEO friendly real estate portal.

== Description ==

Use this plugin to easily install the SEO RETS WordPress plugin on your real estate website.

== Installation ==

1. Upload the `seo-friendly-rets-idx` folder to the `/wp-content/plugins/` directory, or upload the .zip file through the Add New section of the Plugins menu
2. Activate the plugin
3. You will now have SEO RETS installed and the installer plugin will be automatically removed
